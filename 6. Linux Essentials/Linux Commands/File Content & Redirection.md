|Command|Description|
|---|---|
|`cat file.txt`|Displays contents of a file|
|`cat /etc/passwd`|Displays user account details|
|`cat /etc/shadow`|Shows password hashes (restricted)|
|`echo "Hello" > file.txt`|Overwrites file.txt with "Hello"|
|`echo "Hi" >> file.txt`|Appends "Hi" to file.txt|
|`echo "There is best" >> file.txt`|Appends line to file.txt|
