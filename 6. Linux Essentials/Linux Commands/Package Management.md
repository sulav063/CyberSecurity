|Command|Description|
|---|---|
|`sudo apt update`|Updates repository index|
|`sudo apt upgrade -y`|Upgrades all installed packages|
|`sudo apt full-upgrade -y`|Upgrades and handles dependencies|
|`sudo apt install package-name`|Installs a package|
|`apt search nessus`|Searches for packages named ‘nessus’|
|`dpkg -i package.deb`|Installs a `.deb` package manually|
|`sudo nano /etc/apt/sources.list`|Opens software source list file for editing|
