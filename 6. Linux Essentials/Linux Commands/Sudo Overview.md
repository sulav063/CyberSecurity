### Sudo Basics
```
sudo command        # Run command with superuser privileges
sudo -i             # Switch to root environment
sudo -l             # List allowed sudo commands
sudo -u user cmd    # Run command as specified user
```

****
### Sudo Configuration
```
sudo visudo         # Safely edit sudoers file
sudo -k             # Invalidate sudo timestamp
sudo !!             # Re-run last command with sudo
```
