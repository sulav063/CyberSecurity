## vi  
Opens the file in the **vi** text editor.  
Press `i` to insert, `Esc` to exit insert mode, and `:wq` to save and quit.  
**Command:** `vi filename.txt`
![[vi1.png]]
![[vi2.png]]

![[nano2.png]]
