## Frontend (Client-Side)
Frontend is everything the user sees directly on the screen when visiting a website.
### Technologies Used
- HTML
- CSS
- JavaScript
### Frontend Responsibilities
- Designing user interface
- Making pages interactive
- Ensuring responsiveness
- Form validation (client-side)
- Fetching data from backend
### Tools/Frameworks
- React
- Angular
- Vue
- Bootstrap
- Tailwind CSS

---
## Backend (Server-Side)
Backend is the behind-the-scenes logic that processes user actions, manages data, and generates dynamic responses.
### Technologies Used
- PHP
- Python (Django, Flask)
- NodeJS
- Java (Spring)
- C# (.NET)
### Backend Responsibilities
- Authentication & authorization
- Processing forms
- Running logic
- Connecting to database
- Maintaining sessions
- Storing & retrieving data

---
## Frontend vs Backend
|Feature|Frontend|Backend|
|---|---|---|
|Visibility|Visible to users|Not visible|
|Languages|HTML, CSS, JS|PHP, Python, NodeJS|
|Runs on|Browser|Server|
|Focus|User experience|Data and logic|
|Data access|No direct DB access|Full DB access|
|Role|Presentation layer|Business logic layer|
|Security risk|XSS|SQL injection|
