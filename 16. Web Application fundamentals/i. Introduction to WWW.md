## Definition of World Wide Web
The **World Wide Web (WWW)** is a huge collection of digital documents, multimedia, and resources that are connected through hyperlinks and accessed using the Internet.  
It uses standard technologies like HTML, URLs, HTTP/HTTPS to allow users to view and interact with information on websites.

Simple definition:
The WWW is a system that allows users to access webpages and online resources through web browsers.

---
## Internet vs World Wide Web (WWW)

| **Criteria**             | **Internet**                                                                       | **World Wide Web (WWW)**                                                                                |
| ------------------------ | ---------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------- |
| **Basic Meaning**        | A huge global network connecting millions of computers and devices.                | A vast collection of web pages, websites, multimedia, and online resources accessible through browsers. |
| **What It Is**           | Physical infrastructure + networking system.                                       | A digital information system built on top of the Internet.                                              |
| **Foundational Concept** | Network of networks.                                                               | System of interlinked hypertext documents.                                                              |
| **Who Created It**       | Developed by the U.S. Department of Defense (ARPANET, 1969).                       | Invented by Tim Berners-Lee in 1989.                                                                    |
| **Main Components**      | Routers, switches, cables, satellites, servers, network devices.                   | Websites, web pages, web servers, URLs, hyperlinks.                                                     |
| **What It Includes**     | Email, FTP, VoIP, IPTV, instant messaging, cloud services, WWW, online games, etc. | Only web pages, websites, online content viewed through browsers.                                       |
| **Protocols Used**       | Many protocols: TCP/IP, UDP, ICMP, SMTP, FTP, Telnet, DNS, etc.                    | Mainly HTTP and HTTPS protocols.                                                                        |
| **Type of System**       | Hardware + software system.                                                        | Purely software and information system.                                                                 |
| **Access Method**        | Using any Internet-enabled device and any internet-based service.                  | Using a web browser (Chrome, Firefox, Safari, Edge).                                                    |
| **Nature of Data**       | Raw data transfer between devices.                                                 | Formatted information (HTML, images, videos, scripts).                                                  |
| **Purpose**              | To provide global connectivity and communication between machines.                 | To provide access to linked information and online content.                                             |
| **Dependency**           | The Internet does NOT depend on the [WWW](http://www/).                            | The WWW COMPLETELY depends on the Internet.                                                             |
| **Scope**                | Massive, includes many services.                                                   | Smaller subset of the Internet.                                                                         |
| **Example Activities**   | Sending email, video calling, downloading files, using FTP, online gaming.         | Browsing YouTube, reading Wikipedia, using Facebook, viewing web pages.                                 |
| **Data Format**          | Any format (binary, packets, data streams).                                        | HTML documents, CSS, JavaScript, media files.                                                           |
| **Communication Type**   | Device-to-device communication.                                                    | User-to-website communication.                                                                          |
|**Security Technologies**|Firewalls, IPsec, routing rules, VPNs.|HTTPS, SSL/TLS, cookies, sessions.|

---
## Components of the WWW
- **Web Browser** – Software that displays websites (Chrome, Firefox, Edge).
- **Web Server** – Computers that store and deliver web pages (Apache, Nginx).
- **Protocols** – Rules for communication (HTTP, HTTPS).
- **URLs** – Unique addresses for web resources.
- **Hyperlinks** – Connect webpages.
